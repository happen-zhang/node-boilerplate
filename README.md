# node-boilerplate #

Node.js Web开发基础框架模板。

## 部署 ##

### 克隆模板到本地 ###

```
git clone https://github.com/happen-zhang/node-boilerplate.git myapp
```

### 安装 ###

```
cd myapp

npm install
```

### 修改配置 ###

`/config/config.js`文件进行配置

### 运行 ###

```
// 开发环境
NODE_ENV=development node index.js
```

## 第三方库 ##

* express：3.x版本；

* knex、bookshelf：ORM工具；

* when：promise支持；

* handlerbars：视图模板支持；

* mocha：测试框架；

* gulp：自动任务构建工具；

* lodash：js扩展；
